# AI Quiz – Frontend Assignment

A React + TypeScript + Tailwind CSS implementation of an AI‑themed quiz interface inspired by the provided Figma screens. The UI focuses on a clean quiz card layout with gradients, soft shadows, and subtle AI branding elements to match the design expectations of the assignment. [file:2]

## Live Demo

- Deployed on Vercel: https://ai-quiz-frontend-assignment-blie.vercel.app/ [web:9]

## Tech Stack

- React (with Vite as the build tool)
- TypeScript for type safety
- Tailwind CSS for utility‑first styling and responsive layout [web:35][web:38]

## Features

- Multi‑question quiz flow with next/previous navigation.
- Progress bar that updates based on the current question.
- AI‑styled quiz card with gradients, badge, and animated details.
- Result screen showing percentage score and total correct answers.
- Simple state management using React hooks (`useState`). [web:8][web:14]

## Folder Structure

- `src/App.tsx` – Main application and UI layout for quiz and results.
- `src/data.ts` – Quiz questions and answer options.
- `src/types.ts` – TypeScript types for questions and options.
- `src/index.tsx` / `main.tsx` – Application entry point.
- `src/index.css` – Tailwind CSS setup and base styles.
